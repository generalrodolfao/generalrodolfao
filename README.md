## Projetos & Realizações

### 🧠 Multi-Agent Systems
Squads de agentes especializados orquestrando fluxos completos com **LangGraph + MCP** — pesquisa web, design visual, revisão crítica e publishing automatizado.

### 🔍 RAG em Produção
Evolução de *naive → advanced → agentic* RAG. Embeddings, retrieval híbrido, reranking. ChromaDB, pgvector, Pinecone, Qdrant.
> [`rag_corporativo`](https://github.com/generalrodolfao/rag_corporativo) · [`langchat_open`](https://github.com/generalrodolfao/langchat_open)

### 🚀 OpenRoute
Roteamento inteligente entre LLMs (local vs API) com cache semântico. Redução de até **60%** em custo de inferência com fallback automático entre modelos.

### 🤖 Agentes de Software
**Spec-Driven Dev** com Claude Code & Open Code — agentes que geram código production-grade a partir de especificações: frontend, backend, testes e documentação.

### 🎨 Geração de Imagens Agentic
Agentes que interpretam briefings, definem direção de arte e geram visuais com Flux, Stable Diffusion, DALL·E e HTML→screenshot para redes sociais.

### 📊 Observabilidade LLM
**LangFuse + RAGAS** — tracing ponta a ponta, LLM evals (faithfulness, context precision/recall), monitoramento de drift semântico e custo por query.

### 🏗️ Arquitetura AI-Driven
Batch, streaming (CDC) e APIs conectando sistemas corporativos a LLMs em tempo real. OpenMetadata para governança e linhagem de dados.
> [`airflow-blackops`](https://github.com/generalrodolfao/airflow-blackops) · [`infra_dados`](https://github.com/generalrodolfao/infra_dados) · [`dw_blackops`](https://github.com/generalrodolfao/dw_blackops)

### 📚 Educação & Liderança
**+7.000 profissionais capacitados** em Data & AI na América Latina. Liderança de squads multidisciplinares de até 40 pessoas. Atuação com C-levels.
> [`standards`](https://github.com/generalrodolfao/standards) — Aurum: validador inteligente de qualidade de projetos
